Recent Projects

Virtual Pet (ThortonBurg)

A simple Java console application utilizing methods and properties to simulate a single virtual pet. Meet ThortonBurg. He doesn't do much, but you can still feed, water or play with him. And, all of his parts work thanks to the magic of Test Driven Development and unit tests.

Virtual Pets [running] Amok

Thanks to class inheritance and interface implementations we now have so many virtual pets that they need a shelter. Don't worry, you can still interact will one or all of them. Please help feed, water and clean up after them too or they will get sick. Luckily all of the pieces work since Test Driven Development was used from the beginning and all parts have units tests.

Reviews Site

Utilizing Core Java along with Spring MVC, Spring Boot, and Thymeleaf, this project allows a user to scroll through and select a review to read at their leisure. Special thanks to baconipsum.com for the added filler.

Portfolio Site

This was a portfolio site highlighting work done recently. This site utilized grid, flexboxes, and CSS styling.

Reviews Site - Full Stack Edition

This was a continuation of the previous Reviews site but now the 'Reviews' are stored and retrieved in an H2 database and accessed using Spring JPA. The change to using JPA meant that relationships between the data objects had to be created and handled appropriately. Also, 'categories' were added, with appropriate relationships and navigation options.
