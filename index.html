Virtual Pet (ThortonBurg)

A simple Java console application utilizing methods and properties to simulate a single virtual pet. Meet ThortonBurg. He doesn't do much, but you can still feed, water or play with him. And all of his parts work thank to the magic of Test Driven Development and unit tests.

Virtual Pets [running] Amok

Thanks to class inheritance and interface implmentations we now have so many virtual pets that they need a shelter. Don't worry, you can still interact will one or all of them. Please help feed, water and clean up after them too or they will get sick. Luckly all of the pieces work since Test Driven Development was used from the beginning and all parts have units tests.

Reviews Site

Utilizing Core Java along with Spring MVC, Spring Boot, and Thymeleaf, this project allows a user to scroll through and select a review to read at their leisure. Special thanks to baconipsum.com for the added filler.
